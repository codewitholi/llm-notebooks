# README.md

## Getting started

### Requirements

- Python 3.10.11
- cuda
   - Following instructions for your OS / distro

Minimum Specifications for GPU Training
- Free disk space: 24GB
- GPU: 3080 GPU, 10GB VRAM


### Setting up

```bash
$ python3 -m venv .venv
$ source .venv/bin/activate
$ (.venv) pip install -r requirements.txt
```

Run the notebook

```bash
$ jupyter-lab
```
